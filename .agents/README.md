# Agent Workspace

This directory stores durable project memory for AI-assisted development. It is part of the source of truth, not scratch space.

Use it to preserve operating rules, current state, decisions, knowledge, experiments, checklists, and repeatable playbooks. Keep content concise, auditable, and free of private reference names or copied external text.

## Structure

- `program.md`: operating program for agents.
- `current.md`: current project state and next safe task.
- `index.md`: map of agent-facing files.
- `log.md`: append-only chronological work log.
- `decisions.md`: durable architecture decisions.
- `roadmap.md`: committed product capabilities and delivery order.
- `knowledge/`: maintained synthesis.
- `raw/`: sanitized source register and intake notes.
- `checklists/`: acceptance gates.
- `playbooks/`: repeatable workflows.
- `experiments/`: bounded experiment records.
- `reviews/`: review notes and release readiness records.
