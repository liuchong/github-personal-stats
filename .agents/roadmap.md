# Roadmap

## Delivery Order

1. Agent operating framework.
2. Rust workspace, 1PL license, CI, and foundational documentation.
3. Typed configuration and data clients.
4. Aggregation logic for profile metrics, languages, streaks, repositories, gists, status, and coding activity.
5. Unified dashboard SVG renderer and individual card renderers.
6. CLI and release-binary Action.
7. HTTP server, container image, and deployment examples.
8. README section update mode for coding activity.

## Product Commitments

- Default output is a single dashboard SVG.
- Individual cards remain selectable by parameter.
- Width, height, layout, theme, and content sections are configurable.
- Action users consume prebuilt release binaries rather than compiling during workflow runs.
- Local CLI and HTTP deployment are first-class paths.
