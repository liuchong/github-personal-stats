## Lessons To Preserve

- Use a single generated dashboard by default to avoid README layout drift.
- Keep individual card output available for users who need separate assets.
- Download release binaries in Actions instead of compiling during consuming workflows.
- Separate durable knowledge from raw research notes.
- Keep experiments bounded with metrics and keep/discard criteria.
- Prevent private reference names and copied source text from entering repository content.
