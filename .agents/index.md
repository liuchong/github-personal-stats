# Agent File Index

## Root

- `AGENTS.md`: mandatory project instructions for all agents.

## Operating State

- `.agents/program.md`: workflow for startup, implementation, review, experiments, and stop conditions.
- `.agents/current.md`: current state and next safe task.
- `.agents/log.md`: chronological record.
- `.agents/decisions.md`: durable architecture decisions.
- `.agents/roadmap.md`: planned product capabilities.

## Knowledge

- `.agents/knowledge/architecture.md`: system structure and boundaries.
- `.agents/knowledge/api-contracts.md`: API and data contract notes.
- `.agents/knowledge/rendering.md`: SVG and text rendering rules.
- `.agents/knowledge/deployment.md`: CLI, Action, server, release, and hosting notes.
- `.agents/knowledge/compatibility.md`: compatibility concepts and parameter mapping notes.

## Intake

- `.agents/raw/references.md`: sanitized reference categories and lessons learned.

## Checklists

- `.agents/checklists/implementation.md`: implementation readiness and completion checks.
- `.agents/checklists/review.md`: review checklist.
- `.agents/checklists/release.md`: release checklist.
- `.agents/checklists/security.md`: security checklist.
